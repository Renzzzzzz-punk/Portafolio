# Interactive Skeleton Portfolio Resume UI

A creative and modern portfolio resume UI featuring **Mr Skeleton** as a web developer.  
This project includes a clean two-column profile layout, social media buttons, work experience timeline, call-to-action buttons, and a stylish blue-themed design.

## Overview

This project is a creative portfolio/resume interface designed for web developers, designers, and creative coding showcases.  
It uses a playful skeleton avatar concept while keeping the layout professional, clean, and suitable for portfolio websites or resume landing pages.

## Features

- Modern portfolio resume layout
- Skeleton character profile section
- Web developer title and introduction area
- Social media icon buttons
- Work experience timeline
- Download CV button
- Contact button
- Clean blue and white color theme
- Responsive portfolio UI inspiration
- Suitable for HTML, CSS, JavaScript, React, or frontend practice
